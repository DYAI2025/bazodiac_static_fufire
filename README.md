## Dateien

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `package.json`

## Lokal starten

Direkt öffnen:

```txt
index.html
```

Oder aus dem Ordner:

```bash
python3 -m http.server 5173
```

Dann:

```txt
http://localhost:5173
```

Oder:

```bash
npm run dev
```
